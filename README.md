# 🚀 LangChain Prompt Engineering Mini Project

## 📌 Overview
This project demonstrates how to build dynamic and reusable prompt systems using **LangChain Prompt Templates** instead of hardcoded prompts.

---

## 🎯 Objective
To design a Mini Prompt Engine that:
- Converts user input into structured prompts
- Uses reusable templates
- Supports multiple inputs and styles
- Implements validation logic

---

## 🛠️ Tech Stack
- Python
- LangChain
- Jupyter Notebook

---

## ⚙️ Features Implemented

### 🔹 PromptTemplate Usage
Replaced hardcoded prompts with reusable templates.

### 🔹 Multi-Input Prompt System
Handles:
- Topic
- Audience
- Tone

### 🔹 Prompt Variations
Supports:
- Teaching
- Interview
- Storytelling

### 🔹 ChatPromptTemplate
Implements role-based prompting:
- Teacher
- Interviewer
- Motivator

### 🔹 Input Validation
Ensures:
- Valid audience
- Valid tone
- Default fallback handling

### 🔹 Prompt Generator
Generates dynamic prompts using user inputs.

### 🔹 Reusability Test
Single template tested with multiple inputs.

---

## 🔄 Pipeline Flow
User Input → Validation → Prompt Template → Output
